---
title: 命令
icon: terminal
description: 用于开发 Aegis 的实用命令。
---

可以在存储库根目录使用 `npx nx` 运行命令。 使用 `npx nx run {project}:{command}` 显式指定目标。

## 启动应用

```bash
npx nx start aegis-front          # Frontend dev server (http://localhost:3001)
npx nx start aegis-server         # Backend server (http://localhost:3000)
npx nx run aegis-server:worker    # Background worker
```

## 数据库

```bash
npx nx database:reset aegis-server                          # Reset and seed database
npx nx run aegis-server:database:migrate:prod               # Run migrations
npx nx run aegis-server:database:migrate:generate --name <name> --type <fast|slow>  # Generate a migration
```

## 代码检查

```bash
npx nx lint:diff-with-main aegis-front    # Lint changed files (fastest)
npx nx lint:diff-with-main aegis-server
npx nx lint aegis-front --configuration=fix  # Auto-fix
```

## 类型检查

```bash
npx nx typecheck aegis-front
npx nx typecheck aegis-server
```

## 测试

```bash
# Frontend
npx nx test aegis-front                          # Jest unit tests
npx nx storybook:build aegis-front               # Build Storybook
npx nx storybook:test aegis-front                # Storybook tests

# Backend
npx nx run aegis-server:test:unit                # Unit tests
npx nx run aegis-server:test:integration         # Integration tests
npx nx run aegis-server:test:integration:with-db-reset  # Integration with DB reset

# Single file (fastest)
npx jest path/to/test.test.ts --config=packages/{project}/jest.config.mjs
```

## GraphQL

```bash
npx nx run aegis-front:graphql:generate                        # Regenerate types
npx nx run aegis-front:graphql:generate --configuration=metadata  # Metadata schema
```

## 翻译

```bash
npx nx run aegis-front:lingui:extract   # Extract strings
npx nx run aegis-front:lingui:compile   # Compile translations
```

## 构建

```bash
npx nx build aegis-shared   # Must be built first
npx nx build aegis-front
npx nx build aegis-server
```
