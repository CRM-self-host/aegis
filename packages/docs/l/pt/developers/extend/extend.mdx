## Visão geral

O Aegis foi projetado para ser extensível. Use nossas APIs, webhooks e o framework de aplicativos para integrar-se às suas ferramentas existentes e criar funcionalidades personalizadas.

## O que você pode fazer

* **APIs**: Consulte e modifique seus dados de CRM programaticamente usando REST ou GraphQL
* **Webhooks**: Receba notificações em tempo real quando eventos ocorrerem no Aegis
* **Apps**: Crie aplicativos personalizados que expandem as capacidades do Aegis
